NAVBAR

This is an effort to build a css + html navbar code that can be copied / pasted into projects.
Because one day I was a newbie and I wanted to do this kind of thing, without dealing with javascript / sass / preprocessors / frameworks mambo jambos and I couldn't find simple solutions anywhere.

I tried to make everything readable, clear with minimum setup.

The navbar features:

  • Responsive - the menu collapse/expand at screen width 678px breakpoint
  • Anchor element a
  • Not link element p
  • Dropdown elements with class .dropdown and ul li a elements
  • A simple integrated form
  • CSS variables customize fonts, colors and sizes
  • Helper classes .at-right and .at-center to reposition the menu

Observe that:

Every .dropdown div must have the following elements and structure, and the "toggle-1" must be a different pair for every set.

           <div class="dropdown">
             <label for="toggle-1">First Dropdown</label>
             <input type="checkbox" id="toggle-1" />
             <ul>
               <li><a href="#">Dropdown 1st link</a></li>
               <li><a href="#">Dropdown 2nd link</a></li>
               <li><a href="#">Dropdown 3rd link</a></li>
               <li><a href="#">Dropdown 4th link</a></li>
             </ul>
          </div>

This way each label opens it's exclusive/addressed dropdown list.
